# 迈迈语音助手

 > 以上海散爆网络公司旗下游戏——《云图计划》，角色“迈迈”为原型制作的语音交互助手,是橱力向作品。

 > 主要功能通过语音交互，简单操作操作系统，由于是开源程序可以高度自定义功能。

 > 项目 __除__ snowboy离线唤醒模块 和 API调用模块（天气 和 语音转文字 asr） 以及 美术资源（音频，视频，模型等）等,之外均默认使用 __CC-0 开源协议__，作者放弃一切权利。美术资源权利归上海散爆网络所有。
